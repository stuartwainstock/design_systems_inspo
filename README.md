# Design Systems Inspiration

A personal bookmark list of design systems, style guides, Storybooks, and books I return to when researching or building. It’s shared publicly so anyone can use it—not a comprehensive directory, just resources I’ve found genuinely useful.

**For visitors:** Browse by section, search the page, or filter by **Tags**. Notes explain why something landed here when it isn’t obvious from the name. Gaps and dead links happen; this reflects my reading list, not an official index.

**For me:** Add when I’d send someone the link again. Prefer documented, maintained systems—or older references that still teach something.

**Tags:** `tokens` · `a11y` · `content` — comma-separated when more than one applies. Leave blank when none fit yet.
